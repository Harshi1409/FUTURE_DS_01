# 📊 Superstore Sales Analysis

## 📌 Project Overview
This project analyzes sales data from a superstore dataset to understand regional performance and generate business insights.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📈 Key Analysis
- Sales by Region
- Top performing categories
- Data exploration using pandas

## 📊 Visualization
A bar chart is used to compare sales across different regions.

## 🧠 Insights
- West region has the highest sales.
- South region has the lowest sales.
- East region is performing well.
- Sales distribution is uneven across regions.

## 📁 Files
- analysis.ipynb → Main analysis notebook

## 🚀 Conclusion
This analysis helps identify strong and weak performing regions, which can help businesses make better decisions.
